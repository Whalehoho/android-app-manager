# Android App Manager

## Features
- [x] Shortcuts to launch apps through pattern drawing
- [ ] Change app icons
- [ ] Change app titles
- [ ] App lock
- [x] Floating button


Android stores patterns using SharedPreferences at:
/data/data/com.kc123.appmanager/shared_prefs/pattern_prefs.xml
